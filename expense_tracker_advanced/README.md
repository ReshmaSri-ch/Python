# 💸 Personal Expense Tracker (Python)

A command-line and GUI-based Expense Tracker built in Python using SQLite, with CSV export, category/month filters, and chart visualizations.

## ✅ Features

- Add, store, and manage expenses
- SQLite database backend
- Category-wise summary reports
- Month & Year-based filtering
- Export to CSV
- Optional GUI using Tkinter
- Visual charts with matplotlib

## 🛠 Tech Stack

- Python 3.x
- SQLite (`sqlite3`)
- Tkinter (GUI)
- matplotlib (charts)
- CSV, datetime

## 🚀 Getting Started

### 🔹 Install dependencies
```bash
pip install matplotlib
```

### 🔹 Run CLI
```bash
python main.py
```

### 🔹 Run GUI
```bash
python gui.py
```
